search syntax
betänkandet
single term — Swedish stemmer matches betänkande, betänkanden, betänkandet
minister regering
multiple terms — documents containing any term match; ranked by relevance (BM25)
☑ fuzzy
tolerate typos up to edit-distance 2. Use this when unsure of spelling — betänkadet (typo) → finds betänkandet
☑ phrase
match the exact phrase in order — e.g. alkoholmonopolets framtid only finds those words adjacent
Note: AND/OR/NOT in the search box don't work — Lance FTS treats them as terms. Use the fuzzy toggle for typos and the filters below to narrow results.
filters
language
document language (ISO 639-1: sv, en) — exact match
namn
event name from the Riksarkivet metadata CSV — substring match (e.g. alkoholmonopolet matches the full title)
referenskod
archival reference code (e.g. SE/RA/1207/07/B 4/13) — substring match, partial codes work
extraid
internal Riksarkivet identifier — exact match
Filters combine with AND, applied after the FTS query. Spell Swedish words correctly — typos like betänkadet (missing n) won't match.
